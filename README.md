# 91porn

bash 版本视频下载，目前仅支持单个视频下载

#软件环境(Ubuntu/Debian system)</br>
  1.sudo apt-get install wget </br>
  2.sudo apt-get install curl</br>

#使用方法</br>

sh 91.sh -u "url" -p "ip:port" -s "ip:port"</br>

后面2个参数是可选的，URL 是必填</br>
